# 支持的格式说明（Markdown → Word）

本文档说明当前导出到 Word 支持的 Markdown 格式与写法示例。

## 支持范围概览

- 标题：H1~H4（`#` 到 `####`）
- 文本样式：加粗、斜体、删除线、高亮、上下标、行内代码
- 列表：有序/无序列表，支持嵌套
- 表格：Markdown 表格语法（导出默认居中）
- 数学公式：行内 `$...$`，块级 `$$...$$`（居中显示、右侧自动编号）
- 引用标记：`[1]` 会被识别并出现在预览摘要中

## 语法示例

### 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
```

### 文本样式

```markdown
**加粗**  *斜体*  ~~删除线~~  ==高亮==
X^2^  H~2~O  `行内代码`
```

### 列表

```markdown
- 第一项
  - 嵌套子项
1. 第一项
2. 第二项
```

### 表格

```markdown
| 列1 | 列2 |
| --- | --- |
| A   | B   |
```

### 数学公式

行内公式：

```markdown
勾股定理：$a^2 + b^2 = c^2$
```

块级公式（推荐单独成行）：

```markdown
$$
\int_a^b f(x)\,dx
$$
```

块级公式（同一行写法也支持）：

```markdown
块级公式：$$\int_a^b f(x)\,dx$$
```

### 引用标记

```markdown
这是一个引用标记示例 [1]
```
