# v0.1.0

首个可用版本，聚焦「Markdown -> Word」排版能力与离线桌面发布链路。

## Added

- 提供 Electron 桌面打包能力，并内置 FastAPI 后端，支持离线运行。
- 增加多平台桌面发布流程（macOS / Windows / Linux）与 GitHub Release 工作流。
- 扩展 Markdown 导出能力：列表、表格、数学公式（含块公式编号）、行内样式。
- 增强导出表现：三线表边框优化、数学块居中与空白对齐修正。
- 增加合规文档：隐私政策与 EULA。

## Fixed

- 稳定 Web 预览分页行为，并修正 Turbopack 根目录配置问题。
- 桌面模式下放通 `file://` 场景所需的 CORS 来源，提升本地运行稳定性。
